All cross-domain relationships
From To Relationship Meaning
Architecture Java :IMPLEMENTED_BY Arch patterns point to the Java class or method that implements them
Data Architecture :MAPS_TO Data:Service, Endpoint, Queue and Database each map to their Arch counterpart
Data Java :MAPS_TO Data:DataModel maps directly to the Java:Class that defines the type
Tooling Java :SAME_REPO Tooling:Repository is linked to the Java:Repository of the same repo — only created when both code analysis and git mining have run
Java is the foundation — it carries no dependency on any other domain. Tooling is independent of Architecture and Data and can be mined without running code analysis.
